Responsive Web Design Concepts

Learn how media queries, viewport settings, and CSS units work together to create mobile-friendly websites that adapt to any screen size.

Media Queries

CSS rules that apply styles based on device characteristics like screen width, height, and orientation.

@media (max-width: 768px) { ... }

Viewport

The visible area of a web page. Viewport meta tag controls how content is displayed on mobile devices.

<meta name="viewport" content="width=device-width">

CSS Units

Flexible units like %, rem, vw, vh that scale with screen size instead of fixed pixel values.

width: 100%; font-size: 1.2rem; height: 50vh;

Breakpoints

Specific screen widths where layout changes occur to optimize for different device sizes.

320px, 768px, 1024px, 1200px
